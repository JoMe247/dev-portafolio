# Portafolio de trabajo

Portafolio público para presentar experiencia en desarrollo web sin exponer código privado.

## Proyectos incluidos

- CRM de Mensajería y Documentos desarrollado con Laravel, PHP, MySQL y Twilio.
- Sitio inmobiliario en WordPress con personalización de tema hijo, formularios y WhatsApp.
- Despliegue de aplicación Laravel en Ubuntu Server con Nginx, MySQL y PHP.
- Publicación de proyecto Laravel en Hostinger usando Git, SSH, SMTP y rutas personalizadas.

## Tecnologías

Laravel, PHP, MySQL, JavaScript, HTML, CSS, WordPress, Twilio API, Dompdf, Linux, Nginx, Git y GitHub.

## Nota

Este repositorio no contiene código fuente privado. Su objetivo es mostrar experiencia, tecnologías utilizadas y alcance de proyectos realizados.
